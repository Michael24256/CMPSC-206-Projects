Progress Bar:

To make a progress bar, you will need to do the following

  1. div class="progress"
  2. div class="progress-bar"
  3. Striped - progress-bar-striped
  4. Animated - progress-bar-animated
  5. style - to change width and height

Modal:

To make a modal, you will need to do the following

  1. button type="button", class="btn btn-primary", data-toggle="modal" data-target="example', button
  2. div class ="modal" tabindex="-1", id="demo"
  3. div class="modal-dialog"
  4. div class="modal-content
  5. div class="modal-header"
  6. h5 class="modal-title" ##### h5
